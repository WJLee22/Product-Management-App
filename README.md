# Product-Management-App

Spring Boot 기반 Product 관리 시스템 with Spring Security

## 🔧 기술 스택
- Spring Boot 3.x
- Spring Data JPA
- Spring Security (BCrypt, Role 기반 인가)
- Thymeleaf
- MySQL

## ✅ 주요 기능
- 사용자 회원가입 / 로그인 / 로그아웃
- ROLE_USER: 상품 조회만 가능
- ROLE_ADMIN: 상품 등록/수정/삭제 + 관리자 페이지 접근
- 유효성 검사 및 권한별 버튼 제어
